# Amazon Feedback LLM Project

A Streamlit-based application that analyzes Amazon product reviews using **Gemini LLM** and **RAG (Retrieval-Augmented Generation)** powered by **ChromaDB**.

---

## Overview
This project processes and analyzes customer feedback from Amazon products to extract:
- Complaints  
- Suggestions  
- Positive insights  

The system uses a combination of **data preprocessing**, **LLM-powered text analysis**, and **RAG-based retrieval** to answer user questions interactively.

---

## Key Features

Text cleaning & preprocessing with Python (Pandas, Regex)

LLM analysis using Google Gemini

Vector storage and retrieval with ChromaDB

Streamlit interface for interactive Q&A
